# maven-webapp-gitops

Repo contains Helm chart + ArgoCD Application + Argo Rollouts Canary setup.

Structure:
- chart/: Helm chart with Rollout + services + ingresses
- argo/: Application manifest and AnalysisTemplate

How to use:
1. Install nginx-ingress-controller, Prometheus, Argo Rollouts, Argo CD in your cluster.
2. Update repoURL in argo/app.yaml to point to this repo.
3. Push to Git and let ArgoCD sync.
4. Monitor rollout with: kubectl argo rollouts get rollout <release-name> -w
